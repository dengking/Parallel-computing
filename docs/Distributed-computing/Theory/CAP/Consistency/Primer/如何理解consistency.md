# [一致性](https://baike.baidu.com/item/%E4%B8%80%E8%87%B4%E6%80%A7/9840083?fr=aladdin)

一致性就是数据保持一致，在分布式系统中，可以理解为多个节点中数据的值是一致的（即不会丢失数据）。
